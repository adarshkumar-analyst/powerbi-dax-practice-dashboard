# 📊 Power BI Dashboard (DAX Practice & Revision)

## 📌 Overview

This work is created as part of my **Power BI and DAX revision practice**.
The dataset used is for learning purposes and not based on real-world business data.

The report is divided into two pages:

---

## 📄 Report Structure

### 🔹 Page 1: DAX Measures & Analysis

This page is dedicated to showcasing the **DAX calculations** used in the report.

It includes:

* Running Total
* Month-over-Month (MoM) Growth
* Year-to-Date (YTD) Sales
* Year-over-Year (YoY) Comparison
* Ranking (Top Customers / Products)
* Profit and other calculated metrics

👉 Purpose:
To demonstrate understanding of **DAX logic and calculations**.

---

### 🔹 Page 2: Dashboard (Visualization)

This page presents the **interactive dashboard** built using the calculated measures.

It includes:

* KPI Cards (Sales, Profit, Quantity, Customers)
* Sales Trend (Line Chart)
* Top 5 Customers Analysis
* Category-wise Sales
* Region-wise Distribution
* Slicers for filtering (Category, Region, Date)

👉 Purpose:
To convert data into **meaningful insights through visualization**.

---

## 🧠 DAX Concepts Practiced

* `CALCULATE`
* `FILTER`
* `ALL` / `ALLSELECTED`
* `RANKX`
* `TOPN`
* `SUMX`
* `DATEADD`
* `DATESYTD`
* Running Total
* MoM Growth
* YoY Growth

---

## 🎯 Purpose

This is a **practice-based report** created to revise key Power BI and DAX concepts and improve analytical thinking.
